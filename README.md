## 📂 How to Prepare Your Input Files  

EasyLatex works best with simple text files. Here are some tips:

- Use plain `.txt` files saved with UTF-8 encoding.  
- Structure your text with headings and sections clearly marked. For example, use capital letters or simple labels to separate parts like Introduction, Methods, Results.  
- Avoid complex formatting like tables or images in the input file. You can add these later in the LaTeX file if needed.  
- Keep paragraphs separated by blank lines.  

If you plan to use `.docx` files later, wait for future updates.

## 🔄 Workflow Tips  

EasyLatex includes review rules to keep your LaTeX code clean. After conversion, open the LaTeX file in any LaTeX editor (such as TeXstudio or Overleaf).
